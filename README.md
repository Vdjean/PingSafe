# PingSafe

PingSafe is a mobile-first application that allows users to report suspicious people around them in real-time using their location on the map.

## Features

- **Geolocation**: Users can see their position and nearby “pings” on the map.
- **Ping creation**: Quickly report suspicious people or situations to inform the community.
- **Responsive interface**: Designed for simple and fast mobile use.
- **Real-time notifications**: Alerts for new pings nearby.

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Ruby on Rails
- **Database**: PostgreSQL
- **Deployment**: Heroku
